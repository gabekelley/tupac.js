tupac.js
========

A JavaScript implementation of 2pac time (seconds since Tupac).

Usage
-----

```javascript

var t = new Tupac();

/* get seconds since Tupac */
t.getTupac();
//= 559594403

/* set seconds since Tupac to 0 (time of death) */
t.setTupac(0);

/* get Tupac time as date string */
t.toString();
//= "Fri Sep 13 1996 16:03:00 GMT-0700 (PDT)"

```
